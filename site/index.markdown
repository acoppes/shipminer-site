# Story

The anomaly just appeared in the galaxy and it is eating the Space. Get enough minerals to build a device to save humanity before it's too late.

# Game Cycle 

* Travel to asteroids
* Mine
* Survive threats
* Trade minerals
* Improve ship
* Repeat
